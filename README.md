# Importest for Rails

## Installation
### Install for deno
1. Add the `importest` gem to your Gemfile: `gem 'importest'`
2. Run `bundle install`
3. Run `rails importest:install` or `rails importest:install:deno`

### Install for node
1. Add the `importest` gem to your Gemfile: `gem 'importest'`
2. Run `bundle install`
3. Run `rails importest:install:node`
